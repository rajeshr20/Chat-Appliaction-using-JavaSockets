# Java Chat Application using Socket Programming

## Project Overview
This project is a real-time chat application developed using Java Socket Programming with a WhatsApp-style graphical user interface. The system follows a client-server architecture that allows multiple users to communicate simultaneously over a network.

The project demonstrates the implementation of networking concepts, multithreading, GUI design, and database connectivity using JDBC.

---

## Features
- Real-time communication between multiple users
- WhatsApp-style chat interface
- Online users list
- Emoji support
- Client-server architecture
- Database connectivity using JDBC
- Multiple clients can connect at the same time
- Simple and user-friendly interface

---

## Technologies Used
- Java
- Java Swing (GUI)
- Socket Programming
- MySQL Database
- JDBC (Java Database Connectivity)
- Multithreading
- VS Code / IntelliJ IDE

---

## System Architecture
The application follows Client-Server Architecture:

Client → Sends message → Server → Broadcasts message → Other Clients

The server handles multiple clients simultaneously using multithreading.
